# 📝 Notes App

A modern Notes App built using HTML, CSS, and JavaScript with Local Storage support.

---

## 🚀 Features

- ➕ Add notes  
- ❌ Delete notes  
- 💾 Data saved using localStorage  
- 🔄 Data persists after refresh  
- 📄 Show More / Show Less functionality  
- 🎨 Clean and responsive UI  

---

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript (DOM + LocalStorage)  

---

## 📂 Project Structure

Notes-App
│
├── index.html
├── style.css
└── script.js

---

## 💡 How It Works

- User adds notes using input field  
- Notes are stored in browser localStorage  
- On reload, notes are fetched and displayed  
- If notes > 3, “Show More” button appears  

---

## 🎯 Learning Outcome

- Local Storage usage  
- DOM manipulation  
- Event handling  
- Conditional rendering (Show More feature)  
- Building real-world UI
