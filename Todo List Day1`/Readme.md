
# 📝 Todo List App

A simple and interactive Todo List application built using **HTML, CSS, and JavaScript**. This app allows users to manage daily tasks efficiently with a clean and user-friendly interface.

---

## 🚀 Features

- ➕ Add new tasks  
- ❌ Delete tasks  
- ✅ Mark tasks as completed  
- 🔄 Real-time UI updates  
- 🎯 Simple and clean design  

---

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript (DOM Manipulation)
