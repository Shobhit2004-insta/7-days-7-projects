# 💼 Job Finder App

A modern Job Finder Web App built using HTML, CSS, and JavaScript with search, save, and popup UI features.

---

## 🚀 Features

- 🔍 Search jobs instantly  
- 💾 Save jobs using localStorage  
- 📌 View saved jobs anytime  
- 🪟 Popup (Modal) UI for better user experience  
- 🎨 Modern glassmorphism design  
- ✨ Smooth animations  

---

## 🛠 Technologies Used

- HTML  
- CSS (Glass UI + Animations)  
- JavaScript (DOM + LocalStorage)  

---

## 📂 Project Structure

job-finder/
│
├── index.html
├── style.css
└── script.js

---

## 💡 How It Works

- User searches job using keyword  
- Matching jobs are displayed in popup  
- User can save jobs  
- Saved jobs are stored in localStorage  
- Data persists even after refresh  

---

## 🎯 Learning Outcome

- DOM manipulation  
- Event handling  
- LocalStorage usage  
- Modal (popup) UI implementation  
- Building real-world UI/UX
