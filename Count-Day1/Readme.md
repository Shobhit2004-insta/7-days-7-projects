# 🔢 Counter App

A simple and interactive Counter Application built using HTML, CSS, and JavaScript. The app allows users to increase, decrease, and reset a counter value dynamically.

---

## 🚀 Features

- Increase counter value  
- Decrease counter value  
- Reset counter to default  
- Dynamic color change based on value  
- Clean and responsive UI  

---

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript (DOM Manipulation)

---


---

## 💡 How It Works

Users can interact with buttons to increase, decrease, or reset the counter. JavaScript updates the value in real-time using event listeners and DOM manipulation.
